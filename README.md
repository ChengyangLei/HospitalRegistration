## 开发环境

前端基于javafx，并使用了一些优秀的javafx组件库，例如JFoenix、controlsfx进行图形界面的开发；

后台使用mysql数据库，并利用Hibernate进行数据库的维护和查询。

* Jetbrains Intellij 2018.1 Ultamate Edition
* JavaSE-10
* JFoenix 9.0.4
* controlsfx 9.0
* Hibernate 5.3.0
